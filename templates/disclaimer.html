This presentation was build using a modified version of slideshow (S9) and a template based on Google's HTML5rocks! slide presentation which was created by Marcin Wichary and modified by many people in the Google Chrome team

The presentation template was originally meant to run in Google Chrome. Some experimental features might or might not be coded to run in other browsers for now. (Although you will cry out of happiness when you see so much running in IE with the Chrome Frame plug-in)

Particularities of this HTML5 template for slideshow (S9) are:

  • recent update from HTML5rocks! source presentation including:
    • Google Code Prettify
    • dynamic themes
    • better "in-presentation" notes support
  • slideshow (S9) ehancement:
    • transition slides/logos (see g5syntax samples)
